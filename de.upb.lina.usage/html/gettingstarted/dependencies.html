Required Software

The following plugins need to be installed in order to let the LINA tool suite work correctly:
  • Acceleo 3.5
  • Eclipse Modelling Tools 1.5.1 Juno (4.2)
  • EMF Xcore SDK 1.0.1
  • EMF Compare SDK 1.3.2
  • GMF Tooling 3.0.0
  • GMF Runtime 1.6.0
  • JNA Libary 3.4.0
  • Mylyn EMF/GMF 0.9.1
  • Xtext SDK 2.3.1
  • Eclipse Labs Xtext Testing Utilities 0.9.3
